首页

  • 乐享旅游是一套旅游景区分销的电商系统
  • 包括乐享旅游商城系统、商城后台管理系统、分销商管理系统
  • 前端UI库使用BootStrap、LayUI等技术;网页使用HTML、CSS、JavaScript等技术
  • 后端用到SpringBoot、Spring Security、MyBatis、Spring MVC框架
  • 基础服务：JDK1.8、GIT、MAVEN、GitHub
  • 工具组件：Lombok、Swagger、Hutool、Druid、FastJSON等
  • 数据层：MySQL、redis、阿里云
  • 该项目前台商城包含首页门户、商品分类、新品上新、首页轮播、商品推荐、商品搜索、商品详情、购物车、订单管理、个人中心等模块
  • 该项目后台管理系统包含数据面板、轮播图管理、商品管理、订单管理、会员管理、分类管理、设置管理等模块
  • 该项目分销商管理系统包含分销管理、商铺管理等模块
